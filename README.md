## 🎯 Solution for Problem Statement 13 (AITech)

ContentCraft AI is an advanced marketing intelligence suite designed to solve the challenges of modern creators and businesses. It bridges the gap between raw intuition and data-driven content performance by providing a **unified strategic command center**.

### 🏛️ The Core Objective
To design an intelligent system that assists creators in optimizing digital content strategy using advanced NLP and Deep Learning models. ContentCraft AI analyzes real-time signals, suggests platform-specific optimization, and predicts virality potential.

## 🏗️ Technical Architecture (The Stack)

### **Frontend (Visual Excellence)**
- **React 18 & Vite**: For ultra-fast, responsive performance.
- **Tailwind CSS**: Custom "Military-Grade" aesthetic with consistent design tokens.
- **Framer Motion**: Smooth, premium micro-animations and transitions.
- **Recharts**: Dynamic, reactive data visualization.

### **Backend (Intelligence Root)**
- **Node.js & Express**: High-concurrency API orchestration.
- **MongoDB Atlas**: Highly scalable user and strategy persistence.
- **Multi-Tier AI Failsafe Engine**:
  - **Layer 0 (Primary)**: Groq (Llama-3-70B) — Ultra-high-speed strategic generation.
  - **Layer 1 (Failsafe)**: OpenAI GPT.
  - **Layer 2 (Failsafe)**: Google Gemini.
  - **Layer 3 (Failsafe)**: HuggingFace Mistral.

### **Required Environment Variables**
To run this project locally or in production, you need the following keys in your `.env` file:

```env
PORT=5000
MONGO_URI=your_mongodb_atlas_uri
GROQ_API_KEY=your_groq_key
# Optional Fallbacks
OPENAI_API_KEY=your_openai_key
GEMINI_API_KEY=your_gemini_key
```

## 🚀 Getting Started

1. **Clone the Hub**
   ```bash
   git clone https://github.com/Kshitij-Pandey2605/contentcraft-ai.git
   cd contentcraft-ai
   ```

2. **Initialize Backend**
   ```bash
   cd server
   npm install
   npm start
   ```

3. **Initialize Frontend**
   ```bash
   cd ../client
   npm install
   npm run dev
   ```
